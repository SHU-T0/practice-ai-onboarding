# ナレッジ共有Notion設計

## 目的
研修で得た知識・プロンプト・Tips を組織の資産として蓄積・共有する。

## Notionワークスペース構成

```
AI Builders ワークスペース
├── 📚 AIプロンプト集（データベース）
├── 🛠 トラブルシューティング集（データベース）
├── 📱 プロジェクトギャラリー（データベース）
├── 📖 自分用マニュアル（各自のページ）
├── 📋 研修カリキュラム（参照用）
└── 💬 Q&A アーカイブ（Slackから転記）
```

## 1. AIプロンプト集（データベース）

| プロパティ | 型 | 説明 |
|-----------|-----|------|
| タイトル | Title | プロンプトの概要 |
| カテゴリ | Select | 開発 / 企画 / 調査 / ドキュメント / デバッグ |
| ツール | Select | Cursor / Claude / ChatGPT / Perplexity / v0 |
| プロンプト | Text | 実際のプロンプト文 |
| 結果 | Text | AIの出力結果（良い例） |
| 投稿者 | Person | |
| 評価 | Select | ★★★ / ★★ / ★ |
| タグ | Multi-select | Expo / Supabase / UI / Auth / デバッグ / 企画 |

### 運用ルール
- うまくいったプロンプトは必ず投稿
- 週に1回、全員で新しいプロンプトをレビュー
- 評価★★★のプロンプトは「殿堂入り」としてピン留め

## 4. Slack #ai-builders チャンネル設計

### チャンネルルール
- 質問は遠慮なく（「初歩的」でもOK）
- 解決したら解決方法も投稿（他の人の参考になる）
- うまくいったプロンプトはスレッドでシェア
- 週1回、講師が「今週のTips」を投稿

### 定期投稿テンプレート
**毎週月曜 9:00（自動投稿）:**
```
📱 今週のゴール共有
今週取り組むことを一言で教えてください！
```

**毎週金曜 16:00（自動投稿）:**
```
🎉 今週の成果シェア
今週できたことを教えてください！スクリーンショットも歓迎です。
```

## セットアップ手順

1. Notionワークスペースを作成
2. 上記のデータベースをテンプレートから作成
3. 全員をワークスペースに招待
4. Slack #ai-buildersチャンネルを作成
5. 研修初日に使い方を5分で説明
6. 講師が最初の3つのエントリを投稿（お手本として）
