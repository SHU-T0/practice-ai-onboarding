# AI使い分けガイド

## ツール比較表

| ツール | 得意なこと | 苦手なこと | 使う場面 |
|--------|-----------|-----------|---------|
| ChatGPT | 万能助手、幅広い知識 | 最新情報に弱い | 一般的な質問、ブレスト |
| Claude | 論理的分析、長文理解、コード | 画像生成 | 要件定義、コードレビュー、企画書 |
| Gemini | Google連携、最新情報 | 論理的な推論 | 検索、Google Docs連携 |
| Perplexity | リアルタイム検索、出典付き | 長い対話 | 市場調査、競合分析、ファクトチェック |
| Cursor | コーディングAI | コード以外 | アプリ開発 |
| v0 | Web UI生成 | モバイルアプリ | Webページ作成 |

## 使い分けフローチャート

- アプリのコードを書きたい → Cursor
- Webサイトを作りたい → v0
- 調べ物をしたい → Perplexity
- 企画・分析したい → Claude
- 画像を生成したい → ChatGPT (DALL-E)
- Googleのサービスと連携したい → Gemini

## AIへの質問の基本ルール

- 具体的に指示する
- 役割を与える（「あなたはUXデザイナーです」）
- 出力形式を指定する（「表形式で」「箇条書きで」）
- 段階的に質問する（一度に全部聞かない）
